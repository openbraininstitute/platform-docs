# AWS

## Database

* Engine: PostgreSQL 15
* Instance Class: db.t3.small (2 vCPUs, 2 GB RAM)
* Storage: 50 GB allocated storage
* Multi-AZ: Yes (for high availability)
* Backup: 14-day retention period

## Server

* Launch Type: AWS Fargate (serverless containers, no EC2 instances)
* CPU: 256 CPU units (0.25 vCPU)
* Memory: 512 MB RAM
